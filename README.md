A Jitbit Macro Recorder Keygen

## Build Instructions
* Built with `cmake`
* Requires `Qt` 6.5, `libtommath`, and `libtomcrypt`

1. Change CMAKE_PREFIX_PATH and the "include_directories" commands in CMakeLists.txt
2. `cmake .`
3. `cmake --build . --config Release`
